# ENVS-193DS_winter-2026_final

# General information

This repo is for my final exam from ENVS 193DS. This final exam in ENVS193DS demonstrates proficiency in statistical communication, data visualization, and quantitative analysis in environmental science contexts. 

This repo is my personal repo, @maceyhartmann

# Data and file overview

There is a code folder with my qmd document and html document, and a data 
folder with the following files:
- `Annual_Kelp_All_Years_20250903.csv`: annual kelp biomass data
- `MOPL_nest-site_selection_Duchardt_et_al.csv`: nest site selection data from Duchardt et al.
- `Personal_Data_ENVS194DS.csv`: personal reading habits data collected for ENVS 193DS
