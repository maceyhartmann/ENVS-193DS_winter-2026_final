This repo is my personal repo, @maceyhartmann

# Data and file overview

There is a code folder with my qmd document and html document, and a data 
folder with the following files:
- `Annual_Kelp_All_Years_20250903.csv`: annual kelp biomass data from SBC LTER: Reef: Kelp Forest Community Dynamics: Abundance and size of Giant Kelp (Macrocystis Pyrifera), ongoing since 2000. from SBC LTER: Reef: Kelp Forest Community Dynamics: Abundance and size of Giant Kelp (Macrocystis Pyrifera), ongoing since 2000 for problem 2. Citation: Reed, D, R. Miller. 2025. SBC LTER: Reef: Kelp Forest Community Dynamics: Abundance and size of Giant Kelp (Macrocystis Pyrifera), ongoing since 2000 ver 30. Environmental Data Initiative. https://doi.org/10.6073/pasta/c05388280c63f9effa0c71bf5908be1e. Accessed 2026-03-14.
- `MOPL_nest-site_selection_Duchardt_et_al.csv`: nest site selection data from Duchardt et al. Citation: Duchardt, Courtney; Beck, Jeffrey; Augustine, David (2020). Data from: Mountain Plover habitat selection and nest survival in relation to weather variability and spatial attributes of Black-tailed Prairie Dog disturbance [Dataset]. Dryad. https://doi.org/10.5061/dryad.ttdz08kt.
- `Personal_Data_ENVS194DS.csv`: personal reading habits data I collected for ENVS 193DS
