Problem 1. Research writing

a. Transparent statistical methods

In part 1, they used a simple linear regression, because they are testing whether a single continuous predictor variable (precipitation) predicts a continuous response variable (flooded wetland area). In part 2, they used a Kruskal-Wallis test, because they are comparing the median of a continuous response variable (flooded wetland area) across more than two groups of a single categorical predictor variable (water year classification: wet, above normal, below normal, dry, critical drought).

b. Figure needed

The coworker should make a jitter plot with an overlaid side by side box plot. The x-axis should display water year classification as a categorical variable with five groups (wet, above normal, below normal, dry, critical drought), and the y-axis should display flooded wetland area in m². The boxplot component should show the median as a horizontal line within the box, with the box itself representing the interquartile range and whiskers extending to the full range of the data as these elements directly correspond to the Kruskal-Wallis test’s comparison of medians across groups. The jittered points overlaid on each box should show the underlying raw data for each water year classification, giving the reader a sense of spread and sample size within each group. Using different colors for each water year category would further help a non-statistical audience visually distinguish between the five groups.

c. Suggestions for rewriting

In part 1, we did not find a significant relationship between precipitation and flooded wetland area in the San Joaquin River Delta (simple linear regression: F(df model, df residual) = test statistic, p = 0.11, α = significance level, adjusted R² = effect size). In part 2, we did not find a significant difference in median flooded wetland area across water year classifications (wet, above normal, below normal, dry, and critical drought) (Kruskal-Wallis test: H(df) = test statistic, p = 0.12, α = significance level).

d. Interpretation

One additional variable that could influence flooded wetland area is the number of water management interventions per land parcel (active reflooding events), which is a discrete numeric variable. This variable may influence wetland flooding because infrastructure upgrades and incentive programs that encouraged landowners to retain flooded wetlands longer into the breeding season were associated with the observed increases in flooded wetland area over time.

Problem 3. Data analysis

a. Response variable

In this data set, a 1 indicates that a nest site was actively used by a Mountain Plover for nesting. A 0 indicates an unused site, specifically a randomly selected point within the same colony where no plovers were detected and no nests were observed during surveys in that year.

b. Purpose of study

Visual obstruction (vo_nest) is a numeric continuous variable measuring the height at which vegetation is sufficiently dense to obscure a 2.5-cm wide pole when viewed from a 1-m height, incorporating both vegetation height and density. As visual obstruction increases, the probability of Mountain Plover nest site use would decrease because Mountain Plovers prefer sparse, short vegetation that allows them to detect approaching ground predators while still providing some concealment for a sitting bird. This information was found in paragraph 4 of the discussion section.

Distance to prairie dog colony edge (edgedist) is a continuous variable measuring how far a given point is from the nearest edge of the prairie dog colony. As distance to colony edge increases, the probability of Mountain Plover nest site use would decrease because plovers tend to nest closer to colony edges where complementary resources such as taller vegetation for thermoregulation and predator cover are more accessible for adults traveling with broods. This information was found in paragraph 2 of the discussion section.

d. Run the models

mopl_clean <- mopl |> # create a new object called mopl_clean using mopl
  clean_names() |> # clean up column names so that only lower case letters and underscores are used
  # create binary response variable: 1 = used nest site, 0 = available site
  mutate(used_bin = case_when(
    id == "used" ~ 1,
    id == "available" ~ 0
  )) |>
  # keep only the response variable and two predictors of interest
  select(used_bin, edgedist, vo_nest)

# model 0: null model, no predictors
mopl_model0 <- glm(used_bin ~ 1, # response
                   data = mopl_clean, # data frame
                   family = "binomial") # error distribution

# model 1: saturated model, both visual obstruction and distance to colony edge
mopl_model1 <- glm(used_bin ~ vo_nest + edgedist,  # response vs. edge distance and visual obstruction
              data = mopl_clean, # data frame
              family = "binomial") # error distribution

# model 2: distance to colony edge only
mopl_model2 <- glm( used_bin ~ edgedist, # response vs. edge distance
                    data = mopl_clean, # data frame
                     family = "binomial") # error distribution

# model 3: visual obstruction only
mopl_model3 <- glm( used_bin ~ vo_nest, # response vs. visual obstruction
                    data = mopl_clean, # data frame
                     family = "binomial") # error distribution

j. Interpret your results

The best model as determined by AIC included only visual obstruction as a predictor of Mountain Plover nest site occupancy, suggesting that distance to prairie dog colony edge did not meaningfully improve model fit beyond visual obstruction alone. At zero visual obstruction (completely bare, unobstructed ground), the predicted probability of nest site occupancy is 0.73 or 73% (95% CI: 0.65, 0.80), and as shown in Figure 1, this probability declined steeply and nonlinearly as visual obstruction increased, approaching near-zero probability at visual obstruction values above approximately 6 cm. This negative relationship between visual obstruction and nest site occupancy is biologically consistent with Mountain Plover habitat preferences described in the discussion, where the authors note that plovers select nest sites with sparse, short vegetation in the range of 3–5 cm because the short, dense layer provides some concealment for a sitting bird while still allowing the plover to scan for and detect approaching ground predators. Additionally, Mountain Plovers blend in especially well with bare soil when viewed by aerial predators, further explaining the strong preference for low visual obstruction at nest sites. These findings reinforce the critical role of black-tailed prairie dogs as ecosystem engineers that maintain the open, disturbed habitat structure that Mountain Plovers require for successful nesting.

Problem 4. Affective and exploratory visualizations

a. Comparing visualizations

How are the visualizations different? The affective visualization represents reading data through a hand-drawn tree where each branch represents a reading location (campus, home, coffee shop, outside, other), each leaf represents an individual reading session, and leaf color encodes pages read, making the data experiential and emotionally resonant rather than statistically precise. In contrast, the exploratory visualizations use conventional statistical graphics (a scatterplot showing the relationship between reading duration and pages read, and a boxplot comparing pages read across reading locations). The exploratory visualization prioritizes accurate representation of distributions and relationships while the affective visualization shows aesthetic, emotions, and creativity.

What similarities do you see? Both visualizations have the same core variables from the personal dataset: reading location and pages read. The use of color to distinguish categories or encode magnitude appears across both the affective visualization (leaf colors representing pages read) and the exploratory boxplot (different colors per location).

What patterns do you see? The exploratory boxplot suggests campus reading sessions tend to produce more pages read than home sessions, and the scatterplot shows a positive relationship between reading duration and pages read. These same patterns are present but harder to read precisely in the affective visualization, where the home branch appears to have more leaves clustered at warmer colors (higher pages) than the campus branch, though the non-standard format makes direct comparison less intuitive.

I tried to implement this by adding a more consistent color gradient across leaves in the final affective visualization and using a darker yellow color. b. Designing an analysis

To examine whether the day type (school day or non-school day) predicts the number of pages read per session, I would propose a Wilcoxon rank sum test. My response variable, pages read, is a continuous numeric variable, and my predictor variable, day type, is a categorical variable with two groups (school day and non-school day). A Wilcoxon rank sum test is appropriate here because it compares medians between two groups of a categorical predictor, which directly addresses my research question of whether the type of day influences how much I read. Additionally, given the small sample size in my personal dataset and visual deviations observed in the QQ plots, the data are unlikely to meet the normality assumption required for a two-sample t-test, making the Wilcoxon rank sum test a more appropriate non-parametric alternative that does not assume a normal distribution.

c. Check any assumptions and run your analysis

Check all assumptions

The key assumptions for a Wilcoxon rank sum test include that the response variable is continuous (met by nature of the variable, as pages read is a continuous numeric variable), the predictor variable has two categorical, independent groups (met by nature of the variable (school day or non-school day), observations are independent (met by study design, as each reading session is a separate, independent observation), and that the two groups have similar distribution shapes (must do visual assumption check).

f. Write about your results

I did not find a significant difference in median pages read between non-school days (n = 12, median = 37.5) and school days (n = 22, median = 28) (Wilcoxon rank-sum test: W = 174.5, p = 0.13, α = 0.05), with a small effect size (Cliff’s δ = 0.32).
